# Personal Portfolio Site

My minimalistic, mildly responsive portfolio website featuring a rotating cube and scroll-driven animations.

## Live Site
[**stacktactician.dev**](https://stacktactician.dev)

## Tech Stack
- **Three.js**: Interactive 3D background and cube.
- **Vanilla JS**: No frameworks, lightweight performance.
- **CSS**: Glassmorphism, glitch effects, and responsive design.
